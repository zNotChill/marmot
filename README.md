**Marmot** is a client side mod for Minecraft designed to bring **infinite customizability** to the server side. With Marmot, server developers can implement a wide variety of features the base game doesn't offer from custom screen overlays to camera manipulation and more without ruining the vanilla experience.

## Features

- **Server driven UI customization** - Servers can add or modify client side interfaces and overlays dynamically.
- **Camera manipulation** - Smooth camera transitions, custom angles, and more.
- **Server side keybinds** - Allows the server to change certain keybinds when you join to improve the experience.

## Developers

The Marmot API can be found [here](https://marmot.znotchill.me/) and contains guides for Minestom and the general API.
